В виде дерева

​Визуальное представление групп, атрибутов и значений атрибутов в виде дерева. Редактирование, добавление, удаление непосредственно в дереве.

Drag-and-drop

​Перетаскивайте атрибуты или группы с помощью мыши. Теперь атрибуты и группы можно сортировать с помощью перетаскивания.

Атрибуты категории

Привязка набора атрибутов к категории продукта для быстрого назначения похожим продуктам. Подготовка набора атрибутов категории с помощью Drag-and-Drop или инструмента авто создания.

Дежурные шаблоны

Можно задать шаблон по умолчанию для каждого атрибута и подставить его в значение при добавлении атрибута или атрибута категории к товару.

Объединение дублей

Объединение дублирующихся атрибутов без потери связей между их значениями и товарами.

Пакетная обработка

Пакетная замена разделителя значений, замена регистра букв и клонирование языка.
